== Description ==

Graduate is an educational theme developed by Theme Palace. It is a multipurpose WordPress Educational Theme suitable for school, college, university, digital agency as well as business purposes. It's eye appealing design and highly dynamic features uplifts your organization's reputation. It comes with multiple colors and layouts to support your organization's branding. It comes with a wide variety of options so you can modify Layout, Styling, Featured Slider, Menus, Breadcrumb, Widgets and much more, directly from theme customizer. Graduate can be adapted to any kind of industry â€“ not only educational but also in any kind of academy. This theme is TP Education and WooCommerce Plugin compatible, translation ready and major browsers supportive.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

= Does this theme support any plugins? =
* TP Education
	This is highly recommended plugin for Graduate Theme. This plugin is used for adding custom post type ( Events, Courses, Classes, Excursions, Team, Testimonial ) and it's required meta fields. This plugin is highly compatible to Graduate Theme. There are various sections and pages that requires TP Education Plugin.
	This theme consists TP Education plugin files that has been used to overwrite the default templates to make compatible with this theme.

* Jetpack
	Graduate theme includes support for social sharing feature of Jetpack plugin.

* WP Google Maps
	WP Google Maps plugin is recommended to use google map inside theme.

* Contact Form 7
	Contact Form 7 is recommended to use for contact forms for contact page.

* WooCommerce
	Graduate theme highly supports WooCommerce plugin. This theme consists WooCommerce plugin file that has been used to make design compatible with this theme.
